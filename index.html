Hardware Migration

Evolution of the forge. From prototype to production.

Legacy

Raspberry Pi 5

  • 8GB LPDDR4X RAM
  • SD card boot storage
  • ARM architecture
  • Active cooling required
  • Credit card sized
→
Current

ThinkCentre M75q

  • 14GB DDR4 RAM (+6GB)
  • 238GB NVMe SSD (10x faster)
  • AMD Ryzen 5 PRO 3400GE
  • x86_64 architecture
  • ~5 years old, built to last

Why the Upgrade?

More RAM means more models can run simultaneously. NVMe storage means faster boot times and model loading. The x86 architecture opens doors to better virtualization and broader software compatibility.

Room to grow. Built like a tank. Silent under load.

Total Cost: $100 (used)

Dev Log

Weekly updates from the forge. Hardware, software, and everything in between.

2026-04-28 Hardware

Migration Complete: Pi 5 → ThinkCentre M75q

Successfully migrated the entire Supernova stack from Raspberry Pi 5 to a used Lenovo ThinkCentre M75q. The NVMe boot is noticeably snappier, and having 14GB RAM means we can run larger models without swapping. Cost: $100 for a machine that will serve us well for years.

2026-04-28 System

Website Refresh: New Sections Added

Added Hardware Migration documentation and Dev Log section to the Flux Foundary site. Keeping the same dark theme with orange/purple accents. Mobile-first, always.

2026-04-21 AETHER

Voice Processing Pipeline Prototype

Built initial pipeline for voice message ingestion. Whisper for transcription, local LLM for pattern analysis. All processing stays on-device. Privacy-first architecture validated.

2026-04-14 System

Ollama Model Rotation

Testing kimi-k2.5 as primary model. Good balance of speed and reasoning capability. Keeping mixtral as fallback for comparison testing.

More entries coming weekly. The forge never sleeps.
